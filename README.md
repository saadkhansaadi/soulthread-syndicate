# 🌌 SoulThread Syndicate Storefront

**SoulThread Syndicate** is a high-fidelity, cyberpunk-themed eCommerce storefront designed for elite artists, manga enthusiasts, and tactical gear collectors. Built with a focus on glassmorphism aesthetics, responsive performance, and immersive user experience.

---

## 🚀 Key Features

- **Cyber-Neon UI**: A premium, "Raptr-inspired" design system using glassmorphism, smooth gradients, and interactive micro-animations.
- **3D Hero Engine**: An immersive Swiper.js-powered 3D carousel for featured collections.
- **Dual-Mode Experience**: Toggle seamlessly between **Midnight (Dark)** and **Sunrise (Light)** modes with persistent state management.
- **Pro-Level Navigation**: Fully responsive mobile-first navigation with a functional hamburger drawer and smart layout firewalls.
- **Full eCommerce Flow**: Includes customized pages for:
    - Apparel & Hoodies
    - Digital Art & Manga Chapters
    - Custom Commissions & Brief Submissions
    - Interactive Cart Drawer & Checkout Flow
    - Secure Login & Sign-up Pages

## 🛠️ Technology Stack

- **Frontend**: Semantic HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Modular CSS3 with custom variables and a dedicated `responsive.css` system.
- **Components**: Swiper.js (Sliders), FontAwesome (Icons), Google Fonts (Inter/Outfit).

## 📦 Project Structure

```text
├── Assets/             # Images, Branding, and Media Assets
├── CSS/                # Main Stylesheets (style.css, responsive.css)
├── JS files/           # Logic Hub (main.js, hero-slider.js)
├── index.html          # Homepage (The Syndicate Hub)
└── [Page Name].html    # Specialized Storefront & Content Pages
```

## 🔧 Installation & Usage

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/soulthread-syndicate.git
   ```
2. **Launch**: Open `index.html` in any modern web browser or use a Live Server extension for the best experience.
3. **No Build Step**: This project is built using vanilla technologies, so no `npm install` is required. It's ready for deployment to GitHub Pages or Netlify out of the box.
